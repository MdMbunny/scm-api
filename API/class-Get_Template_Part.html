Class Get_Template_Part

Get_Template_Part.php

Include a file and (optionally) pass arguments to it.

Example usage:

$args = array(
'acf_fc_layout' => 'layout-titolo',
'title' => '',
'tag' => 'h1',
'prepend' => 'no',
'append' => 'no',
'id' => '',
'class' => '',
'attributes' => '',
'style' => '',
);

Get_Template_Part::get_part( '_parts/single/single-title.php', array( 'cont' => $args ));
