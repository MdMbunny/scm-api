Class TGMPA_Utils

Generic utilities for TGMPA.

All methods are static, poor-dev name-spacing class wrapper.

Class was called TGM_Utils in 2.5.0 but renamed TGMPA_Utils in 2.5.1 as this was conflicting with Soliloquy.
