Function arrayToHTML

[GET] Transform array in HTML

Example usage:

$arr = [ 'key'=>'value' ];
$att = [
'container' => 'ul',
'element' => 'li',
'key' => 'strong',
'value' => 'span',
];
$html = arrayToHTML( $arr, 0, 0, $att );

print( $html )

</li>
</ul>
