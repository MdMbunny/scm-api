Function scm_acf_field_fa_preset

[GET] Get FA icon choices by group

Each group contains categories. Each category contains 'name', 'color' and 'choices' attributes.

Example usage:

// Get typography FA icons
$icons = scm_acf_field_fa_preset( 'typography' );
